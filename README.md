# 비트코인 레인보우 웨이브 - 실제 가격 vs 예측 경로

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

비트코인의 실제 가격을 **Bitcoin Rainbow Wave(레인보우 웨이브)** 모델의 8개 예측 선과 비교하는 완전 클라이언트 사이드 인터랙티브 웹 앱입니다.

## 기능

- **8개 레인보우 선**: 실제 비트코인 주간 종가를 8개의 예측 곡선 위에 표시
- **클릭 확대**: 차트를 클릭하면 전체 화면 확대 보기
- **시간 범위 선택**: 전체 / 5년 / 3년 / 1년 / 6개월
- **스케일 전환**: 로그 스케일 ↔ 선형 스케일
- **레이어 제어**: 레인보우 밴드, 예측 경로, ATH 마커, 매수 구간 개별 ON/OFF
- **인터랙티브 툴팁**: 마우스 오버 시 실제가격·예측가격·편차·존 표시
- **사이클 분석 카드**: 각 주요 사이클(2017, 2021, 2025, 2027) 인사이트
- **다크모드 지원**: 시스템 설정 자동 감지
- **CSV 내보내기**: 전체 데이터 다운로드
- **반응형 디자인**: 데스크톱/태블릿/모바일 대응

## 사이클 분석

2025년 예측 ATH는 **$126,000**이었으나, 실제 고점은 **$109,035**(2025년 5월 19일)에 그쳤습니다. 이는 예측 대비 **-13.5%** 하회한 수치입니다. 사이클은 전반적인 레인보우 웨이브 패턴을 따랐지만 진폭은 작았습니다.

### 현재 위치

**$63,947**에서 비트코인은 레인보우 웨이브 모델의 **"매수!"** 존에 위치합니다. 역사적으로 매수 구간은 다음 상승장 이전의 훌륭한 누적 기간을 표시했습니다.
